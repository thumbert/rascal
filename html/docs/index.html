  • hour_beginning_gte A filter for values greater than or equal to a given value in column hour_beginning. Only timezone aware timestamps are accepted, in the format %Y-%m-%dT%H:%M:%S%:z[%Q] for example: 2023-01-01T00:00:00-08:00[America/Los_Angeles].

Competitive retail suppliers energy offers

Scrape offers from the utilities webpage and make them available as an API
  • Get the historical offers for a region (e.g. ISONE), posted between a start and end date. Note that an offer can be out there for several months at a time. Currently only CT offers are available. More will be added over time.
    retail_suppliers/v1/offers/region/isone/start/2022-01-01/end/2022-12-04

ISONE

Monthly settlement prices

Data is available from Jan-2021. The url query string has the following components:
  • source_ptids A comma separated list of source ptids, e.g. source_ptids=4000,4000. If not specified, it is the Mass Hub (4000).
  • sink_ptids A comma separated list of sink ptids, e.g. sink_ptids=4001,4008. If not specified, it is the Maine load zone (4001).
  • buckets filter (optional) a comma separated list of LMP components (lmp, mcc, mlc), e.g. buckets=peak,offpeak. If not specified, it is the peak bucket.
isone/ftr/settle_price/monthly/start/2025-01/end/2025-07?source_ptids=4000,4000&sink_ptids=4001,4008&buckets=peak,offpeak

Daily LMP prices by bucket

CAISO daily LMP prices

Data is available from 2025-12-01. Only DA market for now.

Get daily prices between a start/end date for a given bucket.
The url query string has the following components:

  • node_ids filter (optional) a comma separated list of locations, e.g. node_ids=TH_NP16_GEN-APND,TH_SP15_GEN-APND. If not specified, all nodes are returned (a lot of data.)
  • buckets filter (optional) a comma separated list of bucket names e.g. buckets=Caiso6x16,Caiso1x16H,Caiso7x8,CaisoOffpeak,Atc. If not specified, the atc bucket is returned.
  • components filter (optional) a comma separated list of LMP components (lmp, mcc, mlc), e.g. components=lmp,mcc. If not specified, all three components are returned.
caiso/prices/da/daily/start/2025-12-01/end/2025-12-10?node_ids=TH_NP16_GEN-APND,TH_SP15_GEN-APND&components=lmp&buckets=Caiso6x16,CaisoOffpeak

ISONE daily LMP prices

Calculate daily prices (average over the hourly prices) for of the 3 components (lmp, mcc, mcl), by bucket (standard bucket names are supported). Data available from Jan21 to present for DA and RT markets.

The url query string has the following components:

  • ptids filter (optional) a comma separated list of node ids, e.g. ptids=4000,4001. If not specified, all nodes are returned (a lot of data!)
  • buckets filter (optional) a comma separated list of bucket names e.g. buckets=5x16,2x16H. Currently only the buckets 5x16, 2x16H, 7x8, offpeak and atc are supported. If not specified, the atc bucket is returned.
  • component filter (optional) an LMP component name. e.g. component=lmp. Currently only the components lmp, mcc, mlc are supported. If not specified, the lmp component is returned.
isone/prices/da/daily/start/2024-07-01/end/2024-07-15?ptids=4000,4001&buckets=5x16,2x16H,7x8&component=lmp
isone/prices/rt/daily/start/2024-07-01/end/2024-07-15?ptids=4000,4001&buckets=5x16,2x16H,7x8&component=lmp
isone/prices/da/daily/start/2024-07-01/end/2024-07-15?ptids=4000,4001&buckets=5x16,2x16H,7x8&component=mcc
isone/prices/da/daily/start/2024-07-01/end/2024-07-15?ptids=4000,4001&buckets=5x16,2x16H,7x8&component=mlc

The url query string supports the following filters:

All filters are optional unless otherwise noted. If your query does not include any filters, all rows will be returned. If no data matches your filters, an empty result set will be returned. If the amount of data returned is too large, the server will return an error.

  • market A filter for matching exactly one value in column market. Possible values: "DA", "RT".
  • market_in A list of valid values separated by commas. If the values themselves contain commas, they should be enclosed in double quotes.
  • hour_beginning A filter for matching exactly one value in column hour_beginning. Only timezone aware timestamps are accepted, in the format %Y-%m-%dT%H:%M:%S%:z[%Q] for example: 2026-01-01T00:00:00-05:00[America/New_York].
  • hour_beginning_gte A filter for values greater than or equal to a given value in column hour_beginning. Only timezone aware timestamps are accepted, in the format %Y-%m-%dT%H:%M:%S%:z[%Q] for example: 2026-01-01T00:00:00-05:00[America/New_York].
  • hour_beginning_lt A filter for values less than a given value in column hour_beginning. Only timezone aware timestamps are accepted, in the format %Y-%m-%dT%H:%M:%S%:z[%Q] for example: 2026-01-02T00:00:00-05:00[America/New_York].
  • limiting_facility A filter for matching exactly one value in column limiting_facility.
  • limiting_facility_like A string pattern to be used as a SQL like filter for the values in column limiting_facility.
  • limiting_facility_in A list of valid values separated by commas. If the values themselves contain commas, they should be enclosed in double quotes.
  • facility_ptid A filter for matching exactly one value in column facility_ptid.

  • facility_ptid_gte A filter for values greater than or equal to a given value in column facility_ptid.
  • facility_ptid_lte A filter for values less than or equal to a given value in column facility_ptid.
  • contingency A filter for matching exactly one value in column contingency.
  • contingency_like A string pattern to be used as a SQL like filter for the values in column contingency.
  • contingency_in A list of valid values separated by commas. If the values themselves contain commas, they should be enclosed in double quotes.

  • _limit An optional limit on the number of records returned by the query. If not specified, all matching records will be returned.
