Competitive retail suppliers energy offers

Scrape offers from the utilities webpage and make them available as an API
  • Get the historical offers for a region (e.g. ISONE), posted between a start and end date. Note that an offer can be out there for several months at a time. Currently only CT offers are available. More will be added over time.
    retail_suppliers/v1/offers/region/isone/start/2022-01-01/end/2022-12-04

EPA emission data

  • Get historical emission data by facility. Each power plant is specified by a facility id and an unit id. The query returns data for all units in the facility when there was CO2 emitted. For older periods of time, there may be NOX data only for a given day. This query will not return that data.
    epa_emissions/v1/fac_id/328/start/20170101/end/20170131
  • Get the EPA emission data by power plant. Each power plant is specified by a facility id and an unit id. To get the existing ids, see the [/units] route below.
    epa_emissions/v1/fac_id/328/unit_id/1083/start/20170101/end/20170131
  • Get the unique combination of facility id and unit id in the database.
    epa_emissions/v1/units
  • Get all the facilities tracked in the database. Note that the [FACILITY_NAME] is not unique.
    epa_emissions/v1/facilities
  • Get the daily CO2 emissions for facilities that are subject to the MA GHG 310 CMR 7.74 ruling. The [CO2_MASS] value is in short tons.
    epa_emissions/v1/ma_compliance/start/20170101/end/20171231

Forward marks

  • Get the list of all price curves for one date

  • Get one curve as of a given date
    forward_marks/v2/price/curvename/NG_HENRY_HUB_CME/asofdate/20230727

    forward_marks/v2/price/curvename/OIL_WTI_CME/contract_start/2024-01/contract_end/2024-02/start/2023-01-01/end/2023-07-27

Hydro Quebec hydrological data

  • Get daily water level data for some or all facilities between a start/end date. If no station ids are given, return all of them. Original data is hourly. Perform a daily average, and round the result to two decimals before returning.
    hq/water_level/daily/start/2024-12-04/end/2024-12-08?station_ids=1-2951,1-9698
    hq/water_level/daily/start/2024-12-04/end/2024-12-08

ISONE masked energy offers both DA and RT markets

  • Get all the generation offers between a start/end date.
    The url query string has the following components:
    • masked_asset_ids filter (optional) a comma separated list of ids, e.g. masked_asset_ids=77459.
    isone/energy_offers/da/start/2024-11-01/end/2024-11-03
    isone/energy_offers/rt/start/2024-11-01/end/2024-11-03
    isone/energy_offers/da/start/2024-11-01/end/2024-11-03?masked_asset_ids=77459
  • Get the stack for a comma separated list of timestamps. A timestamp is the number of seconds since the epoch. For example, 1709269200 is 2024-03-01 00:00:00-05:00[America/New_York]. Only the offers of the units that are available are returned, sorted by price.
    isone/energy_offers/da/stack/timestamps/1709269200

NRC unit status

Data from https://www.nrc.gov/reading-rm/doc-collections/event-status/reactor-status/index.html.
  • Get all the generator names in the database.
    nrc/generator_status/unit_names
  • Get the generator status between a start and end date.
    The url query string has the following components:
    • unit-names filter (optional) a comma separated list of unit names, e.g. `unit-names=Calvert Cliffs1,Byron 1`. Note that the unit names need to match exactly the names in the database.
    nrc/generator_status/start/2025-01-01/end/2025-01-04
    nrc/generator_status/start/2025-01-01/end/2025-01-04?unit_names=Calvert Cliffs 1,Byron 1
